# **App Name**: AetherVault

## Core Features:

- Cipher Algorithm Selection: Provides a select from list of encryption algorithms for users to select from.
- Dynamic Parameter Input: Presents dynamic input fields based on the selected encryption algorithm to gather necessary cipher parameters.
- Encryption/Decryption Execution: Executes the selected encryption or decryption algorithm on the client-side using the provided input text and cipher parameters.
- Result Display: Displays the resulting ciphertext or plaintext after successful encryption or decryption, along with a success status.
- Step-by-Step Breakdown (AES/DES): For AES-128 and DES algorithms, provides a detailed, step-by-step breakdown of each round, displayed in a mobile-optimized bottom sheet.

- Accent color: Muted Orange (#E67E22) for calls to action.
- Primary text color: High-contrast light gray (#F5F5F5) for readability.
- Secondary text color: Medium gray (#B0B0B0) for labels and descriptions.
